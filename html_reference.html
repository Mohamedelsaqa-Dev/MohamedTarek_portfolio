HTML Elements Comparison

العنصر / الخاصية وظيفته
<a> (Anchor) رابط ينقل المستخدم إلى صفحة أخرى أو جزء داخل الصفحة
alt (Alternate Text) نص بديل للصور يظهر عند عدم تحميل الصورة أو لتحسين الوصول (Accessibility)
<pre> (Preformatted Text) يعرض النص كما هو، مع المسافات والأسطر الجديدة محفوظة
<b> (Bold) يجعل النص غامق بصريًا فقط، بدون دلالة معنوية
<em> (Emphasis) يشير إلى النص المهم أو المؤكد، غالبًا يُعرض مائل (Italic)
<strong> يشير إلى النص المهم جدًا، عادة يظهر غامق (Bold) ويعطي دلالة معنوية قوية
<mark> يستخدم لتظليل النص (Highlight) لإبرازه
<small> يعرض النص بحجم أصغر من النص العادي
<audio> عنصر لتشغيل ملفات الصوت في الصفحة
<iframe> عنصر لعرض صفحة أو محتوى من موقع آخر داخل الصفحة الحالية
<canvas> مساحة للرسم والرسوميات باستخدام JavaScript (رسم ديناميكي)

HTML Elements & Text Formatting Comparison

العنصر وظيفته / استخدامه
<sub> يعرض النص كـ "Subscript" أي نص صغير أسفل السطر (مثل H2O)
<blockquote> اقتباس طويل من مصدر آخر، عادة يظهر بمسافة بادئة
dir="rtl" يستخدم لجعل اتجاه النص من اليمين لليسار (Right-to-Left)
<abbr> يستخدم للاختصارات ويعرض النص الكامل عند المرور عليه (مثل HTML)
<!-- Comment --> تعليق داخل الكود لا يظهر في الصفحة، يستخدم للتوثيق
<p> فقرة نصية، تستخدم لتقسيم النصوص إلى فقرات
<div> حاوية عامة لتجميع العناصر، لا يظهر تأثير بصري لوحده
<select> قائمة منسدلة (Dropdown) لاختيار قيمة واحدة أو أكثر
<option> عنصر داخل <select> يمثل خيارًا واحدًا يمكن اختياره
word-break: break-word خاصية CSS تستخدم لكسر الكلمات الطويلة داخل العنصر لمنع تجاوز النص
<br> إنشاء سطر جديد (Breakline) داخل النص
<span> عنصر Inline لتنسيق جزء صغير من النص أو لإضافة CSS/JS، لا يضيف خط فاصل
<table> جدول لعرض البيانات في صفوف وأعمدة
<ul> قائمة غير مرتبة (Bulleted List)
<ol> قائمة مرتبة (Numbered List)
<li> عنصر قائمة داخل <ul> أو <ol>
<tr> صف داخل جدول (Table Row)
<td> خلية بيانات داخل صف جدول (Table Data)
<th> خلية رأس داخل جدول (Table Header)، عادة غامقة ومركزية
