Selectors

p {
    color:red;
    }
    /* Element selector targets all p elements */
.box {
    background:blue;
    }
    /* Class selector targets elements with class="box" */
#header {
    background:black;
    }
    /* ID selector targets element with id="header" */

Box Model

div{
    margin:20px;
    }
    /* margin creates space outside element */
div{
    padding:20px;
    }
    /* padding creates space inside element */
div{
    border:1px solid white;
    }
    /* border creates outline around element */

Text Styling

p{
    color:red;
    }
    /* changes text color */
p{
    font-size:20px;
    }
    /* changes text size */
p{
    font-weight:bold;
    }
    /* makes text bold */
p{
    text-align:center;
    }
    /* aligns text */

Display & Layout

div{
    display:block;
    }
    /* block takes full width */
div{
    display:inline;
    }
    /* inline takes content width */
div{
    display:flex;
    }
    /* enables flexbox layout */
div{
    display:grid;
    }
    /* enables grid layout */

Flexbox

.container{
    display:flex;
    justify-content:center;
    }
    /* horizontal alignment */
.container{
    align-items:center;
    }
    /* vertical alignment */
.container{
    gap:20px;
    }
    /* space between items */

Grid

.container{
    display:grid;
    grid-template-columns:1fr 1fr;
    }
    /* creates two equal columns */
.container{
    gap:20px;
    }
    /* space between grid items */

Position

div{
    position:relative;
    }
    /* relative to itself */
div{
    position:absolute;
    }
    /* relative to parent */
div{
    position:fixed;
    }
    /* fixed on screen */

Effects

button:hover{
    background:red;
    }
    /* hover effect */
div{
    transition:0.3s;
    }
    /* smooth animation */
div{
    transform:scale(1.1);
    }
    /* scales element */

Responsive Design

@media (max-width:768px){
    body{
    background:red;
    }
    }
    /* applies style on small screens */

Animations

@keyframes move{
    from{left:0;}
    to{left:100px;}
    }
    /* defines animation */
div{
    animation:move 1s;
    }
    /* applies animation */

css technical revision

المجموعة الخاصية الوظيفة / الشرح مثال عملي الحاجات المتشابهة والفروق
Display Types block العنصر يبدأ في سطر جديد ويأخذ العرض كامل 
div { display:block; }
 - block vs inline: block يبدأ سطر جديد، inline لا
- inline-block: يقبل width/height
- flex/grid: لتخطيط الأبناء
inline العنصر يبقى في نفس السطر مع الآخرين، لا يقبل width/height 
span { display:inline; }
inline-block يشبه inline لكن يقبل width/height 
div { display:inline-block; width:100px; }
flex تخطيط مرن للأبناء داخل العنصر 
.container { display:flex; justify-content:center; }
grid تخطيط شبكي للصفوف والأعمدة للأبناء 
.container { display:grid; grid-template-columns:1fr 2fr; }
Positioning relative العنصر يتحرك نسبة لمكانه الأصلي 
.rel { position:relative; top:10px; }
 - relative vs absolute: relative يتحرك من مكانه الأصلي، absolute يخرج من التدفق
- fixed: ثابت بالنسبة للنافذة
- sticky: يمزج relative و fixed
absolute العنصر يتحرك بالنسبة لأقرب أب position!=static، يخرج من التدفق 
.abs { position:absolute; top:0; left:0; }
fixed العنصر ثابت بالنسبة للنافذة مهما تم التمرير 
.fixed { position:fixed; bottom:0; }
sticky يمزج بين relative و fixed، يلتصق عند نقطة معينة أثناء التمرير 
.sticky { position:sticky; top:0; }
Units px وحدات ثابتة 
div { width:200px; }
 - px ثابت
- % يعتمد على العنصر الأب
- em يعتمد على حجم الخط الحالي للعنصر
- rem يعتمد على حجم الخط الأساسي html
% نسبة من العنصر الأب 
div { width:50%; }
em يعتمد على حجم خط العنصر الحالي 
p { font-size:2em; }
rem يعتمد على حجم الخط الأساسي للصفحة 
p { font-size:2rem; }
Visibility display:none العنصر يختفي ولا يأخذ مساحة 
.hide { display:none; }
 - display:none: يختفي تمامًا
- visibility:hidden: مخفي لكن يأخذ مساحة
- opacity:0: شفاف لكنه موجود
visibility:hidden مخفي لكن يأخذ مساحة في الصفحة 
.hide2 { visibility:hidden; }
opacity:0 شفاف لكن موجود ويأخذ مساحة 
.hide3 { opacity:0; }
Text Styling bold / font-weight تحكم في وزن النص 
p { font-weight:bold; }
 - bold vs font-weight: نفس الوظيفة
- italic / font-style
- underline / text-decoration
italic / font-style تحكم في ميل النص 
p { font-style:italic; }
underline / text-decoration تحكم في خط تحت النص 
p { text-decoration:underline; }
Flex & Grid justify-content / align-items توزيع العناصر داخل flex أو grid 
.container { display:flex; justify-content:center; align-items:center; }
 - justify-content / align-items: توزيع العناصر
- grid-gap / gap: المسافة بين العناصر
- align-self / justify-self: التحكم الفردي لكل عنصر
grid-gap / gap المسافة بين الأعمدة والصفوف 
.container { display:grid; grid-gap:10px; }
align-self / justify-self تحكم فردي في عنصر داخل flex أو grid 
.item { align-self:flex-start; }
Effects box-shadow إضافة ظل حول العنصر 
div { box-shadow:2px 2px 10px rgba(0,0,0,0.3); }
 - box-shadow vs border: الظل والخطوط
- opacity vs visibility vs display:none
- transition يحرك التغييرات بسلاسة
border-radius جعل حواف العنصر دائرية 
div { border-radius:10px; }
opacity تغيير شفافية العنصر 
div { opacity:0.5; }
transition تأثير سلس عند تغيير الخصائص 
div { transition:all 0.3s; } div:hover { transform:scale(1.05); }
Animations @keyframes تعريف حركة أو تأثير متكرر 
@keyframes move { from { left:0; } to { left:100px; } }
 - animation vs transition: transition بدون keyframes
- keyframes لتعريف حركة كاملة
animation تطبيق الحركة مع المدة والتكرار 
div { animation:move 2s infinite; }
transition تغيير سلس للخصائص بدون keyframes 
button { transition:background 0.3s; } button:hover { background:#009879; }
Colors & Backgrounds background-color لون خلفية العنصر 
div { background-color:#009879; }
 - color vs background-color
- gradient backgrounds
- opacity تؤثر على اللون
background-image / gradient خلفية بصورة أو تدريج لوني 
div { background-image:linear-gradient(to right, red, yellow); }
color لون النصوص 
p { color:#333; }
Responsive Design media queries تغيير التصميم حسب حجم الشاشة 
@media (max-width:768px) { body{ font-size:14px; } }
 - media queries للتحكم في التصميم على أجهزة مختلفة
- relative units %, em, rem تجعل التصميم متجاوب
relative units (%, em, rem) حجم العناصر والنصوص يتغير حسب العنصر الأب أو الصفحة 
div { width:50%; font-size:1.2rem; }
